- CHANGELOG -
=============

This is the JMWiki changelog. It lists all non-trivial changes to the project per release,
starting at release 3.0 alpha. The most recent release is on top.


== RELEASE 3.0 Alpha ==

- Core dispatching structure migrated to use Spring framework
- Package structure move from 'vqwiki.*' to 'org.vqwiki.*'
- Improved build routine; lexers are always regenerated.
- Improved build routine; now generates both source and binary release packages.
